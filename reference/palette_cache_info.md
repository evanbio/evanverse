# Get Palette Cache Information

Display information about the current palette cache status, including
whether it's initialized, when it was loaded, and how many palettes are
cached.

## Usage

``` r
palette_cache_info()
```

## Value

List with cache information (invisible)

## Examples

``` r
if (FALSE) { # \dontrun{
# Check cache status
palette_cache_info()
} # }
```
